# Python Port Scanner

Учебный многопоточный TCP-сканер портов на Python.

Проект создан для практики Python, работы с `socket`, многопоточности через `ThreadPoolExecutor`, обработки исключений и базовой сетевой диагностики.

## Возможности
- TCP connect scan
- сканирование диапазона портов или списка портов
- многопоточность
- отображение прогресса в консоли с визуальным баром, процентами, временем и ETA
- определение стандартного имени сервиса по номеру порта
- валидация пользовательского ввода
- настраиваемый таймаут
- вывод результатов после завершения сканирования

## Как работает
Сканер пытается установить TCP-соединение с указанным хостом на каждом порту из заданного диапазона.
Если соединение успешно, порт считается открытым.
Для ускорения проверки используется пул потоков `ThreadPoolExecutor`, поэтому несколько портов сканируются параллельно.

## Требования
- Python 3.10+

Используются только стандартные библиотеки Python:
- `socket`
- `time`
- `concurrent.futures`

## Запуск
```bash
python scanner.py
```

Или с аргументами командной строки:
```bash
# Диапазон портов
python scanner.py --host example.com --ports 1-1024 --threads 100 --timeout 0.5

# Конкретные порты
python scanner.py --host example.com --ports 80,443,22

# Все порты (1-65535)
python scanner.py --host example.com --ports all
```

### Аргументы командной строки:
- `--host`: IP или хост для сканирования (по умолчанию: localhost)
- `--ports`: Диапазон или список портов: диапазон (1-1000), список (80,443,22) или all (по умолчанию: 1-1000)
- `--threads`: Количество потоков (по умолчанию: 50)
- `--timeout`: Таймаут в секундах (по умолчанию: 1.0)

## Важно
Использовать только на своих системах или там, где есть явное разрешение на тестирование.
